# Diploma Project - Dataset & Results
Dataset of test images, result and links for this [diploma project](https://github.com/SkalapEnder/Diploma-Project)

## Dataset
Links:

* [DIV2K](https://www.kaggle.com/datasets/soumikrakshit/div2k-high-resolution-images) - high-resolution images
* [Standard Test Images](https://www.kaggle.com/datasets/saeedehkamjoo/standard-test-images) - a collection of classical test images
* [Set 5 & Set 14](https://www.kaggle.com/datasets/ll01dm/set-5-14-super-resolution-dataset?resource=download) - 
* [Linnaeus 5 dataset](https://chaladze.com/l5/) - a dataset of small images with 256x256 pixels
* [LibPNG](https://www.libpng.org/pub/png/png-sitemap.html#images)
* [Hubble Birthday](https://imagine.gsfc.nasa.gov/hst_bday/)
* [Kodak database](https://r0k.us/graphics/kodak/)
